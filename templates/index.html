Star Predictions Using Machine Learning

We understand that labeling star types can be a long, tedious, heart-breaking process. Our goal is to make your life easier by automating this process. We're using a RandomForestClassifier machine learning model to predict start types based on a variety of features. Below, you'll find more information about the features we're using:

Features Descriptions Example
Temperature Temperature measure in K 2000K to 35000K
Luminosity Average luminosity of star's sun 0.0001w to 300000w
Radius Average radius of star's sun 0.01m to 6000m
Absolute Magnitude The star's absolute magnitude -100 to 100
Spectral Class Classification from asteroid spectral class. O, B, A, F, G, K, M

With these features, we'll be able to accurately predict if a star is a Red Dwarf, Brown Dwarf, White Dwarf, Main Sequence, Super Giant, or Hyper Giant
